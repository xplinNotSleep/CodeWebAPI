## 描述
路径漫游包括录制漫游、绘制漫游、对历史漫游数据的管理功能。
录制漫游：查看历史漫游视频、删除历史漫游记录、录制路径漫游
绘制漫游:手动绘制漫游路径、设置漫游参数，可保存漫游视频
历史漫游:历史漫游记录展示、删除历史漫游、修改绘制漫游
